        Scenarios

        Resident flow Visibility Feedback Constraints

        A. Aina books bulky pickup + RoRo

        Trigger: Finished minor renovation; 6 bulky items. Preconditions: Logged-in, location ON, service area supported.

        1. Open Disposal → Bulky / RoRo; app confirms coverage and shows “What you’ll need”.
        2. Tap Estimate & Slots, upload 2 photos; system auto-classifies item type/volume.
        3. See live price breakdown, earliest slots, and cut-off times.
        4. Select Sat 9–11am; add unit & lift access; toggle concierge/lobby pickup.
        5. Checkout (FPX/CC); get Job ID, checklist, and “Add to Calendar”.
        6. Receive 24h reminder with prep checklist and do/don’t items.
        Alt paths / exceptions
        • No slot → suggest nearest hub or RoRo drop-off.
        • Photo unclear → prompt retake or manual item type.
        • Outside coverage → show nearest region + self-haul options.
        Usability metrics
        • Effectiveness ≥ 95% complete without help
        • Efficiency ≤ 3 mins on 4G
        • Satisfaction ≥ 80% “easy to book”
        UX goals
        • Trust & clarity
        • Low cognitive load
        • Gentle guidance
        Done criteria
        • Email/SMS has slot, prep, reschedule link
        Ops flow Mapping Auditability SLA

        B. Azri triages illegal dumping cluster & assigns job

        Trigger: ≥5 reports cluster at same junction in 24h. Preconditions: Dashboard authenticated; contractor capacities synced.

        1. Open Reports Map; heatmap clusters blink for new spikes.
        2. Open cluster → evidence panel (photos, precision, time, duplicates).
        3. Mark valid → Create Job → Bulk clean-up; zone/volume pre-filled.
        4. System auto-suggests contractor by zone/capacity/proximity; show ETA & route impact.
        5. Dispatch with SLA (8h); reporters get status message.
        6. Contractor uploads before/after photos; job closes after verification.
        Alt paths / exceptions
        • Insufficient evidence → request more photos from top reporters.
        • Capacity full → fallback to nearest available team; recalc ETA.
        • Hazardous waste → escalate to Environmental Unit; change SLA.
        Usability metrics
        • Effectiveness ≥ 90% clusters → jobs
        • Efficiency ≤ 60s median to assign
        • Satisfaction ≥ 80% prioritization clarity
        UX goals
        • Control & speed
        • Confidence in decisions
        • Transparent audit trail
        Done criteria
        • Closed job has geotagged after-photo + SLA met flag + notification sent
